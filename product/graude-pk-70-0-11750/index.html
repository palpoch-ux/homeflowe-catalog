Описание

Холодильник для хранения шуб Graude PK 70.0 предназначен для хранения меховых изделий.

Модель вмещает до 3-х шуб, безопасность размещения которых обеспечивает механический замок.

Все настройки устройства задаются через внешний дисплей.

Прибор представлен в классическом черном исполнении, который имеет расширенные возможности для комбинирования с цветовыми решениями помещения, в котором будет располагаться холодильник.

Для удобства пользователя предусмотрено внутреннее освещение.

Светодиодная подсветка экономична в плане потребления электроэнергии.

Лампочки не нагреваются даже при длительной работе.

Для повышения уровня безопасности модель оснащена специальной сигнализацией.

Она срабатывает на открытую дверцы (время открытия которой превышает допустимый лимит) и на неисправности.

Внутри холодильника для хранения шуб грауде PK 70.0 поддерживаются оптимальные условия, поэтому можно не бояться смешивания запахов.

Шубы размещаются на плечиках и вешаются на перекладину.

Ширина и высота устройства подойдут для различных видов мехов.

Прибор отличает тихая работа, что создает дополнительный комфорт эксплуатации.

Модель можно легко перемещать за счет наличия специальных роликов.

Появляющийся в процессе работы конденсат стекает по дренажной системе в сосуд, который находится сзади.

Такая система абсолютно безопасна для хранения шуб.

Официальная гарантия производителя составляет один год с момента приобретения.
