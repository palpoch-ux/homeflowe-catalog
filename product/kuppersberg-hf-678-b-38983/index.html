Описание

Kuppersberg HF-678-B — встраиваемый электрический духовой шкаф черного стекла с объемом камеры 70 литров, который идеально вписывается в современные кухонные гарнитуры.

Корпус шириной 59,5 см и высотой 59,5 см точно соответствует стандартной нише 56×60 см, утопленные поворотные переключатели с сенсорным дисплеем создают монолитный фасад без выступов.

Внутренняя камера из эмали Crystal Clean устойчива к кислотам и высоким температурам до 300°C, три слоя стекла дверцы нагреваются не выше 65°C снаружи.

Телескопические направляющие на 1 уровне плавно выдвигаются на 50 см, обеспечивая доступ к блюдам без риска ожогов.

Двенадцать режимов нагрева покрывают профессиональные задачи — традиционный нагрев верхним+нижним ТЭНами готовит равномерно при 50-250°C, конвекция с турбовентилятором распределяет тепло для многоуровневой выпечки.

Гриль с внутренним кольцом подрумянивает за 8 минут, макси-гриль+конвекция запекает курицу целиком с хрустящей корочкой.

Разморозка циркулирует комнатный воздух без нагрева, быстрый прогрев достигает 200°C за 5 минут.

Нижний нагрев с конвекцией идеален для пиццы с тонким тестом, верхний ТЭН+вентилятор подсушивает меренгу без пригорания.

Комбинированное управление сочетает механические регуляторы с сенсорным дисплеем — утапливаемые ручки фиксируются в 11 позициях температуры, сенсор активирует таймер и часы.

Многофункциональный таймер отслеживает несколько блюд параллельно, звуковой сигнал предупреждает о готовности.

Доводчик мягко закрывает дверцу без хлопков, съемная конструкция из 3 стекол разбирается без инструментов для мойки.

Гидролизная очистка размягчает жир паром за 25 минут, эмаль не впитывает загрязнения и очищается губкой.

Купперсберг HF-678-B использует ТЭНы суммарной мощностью 3200 Вт — кольцевой верхний 1700 Вт, нижний 1000 Вт, круглый задний 1500 Вт для конвекции.

Класс энергопотребления A гарантирует расход 0,95 кВтч при пиковой нагрузке, класс A+ при повседневном использовании.

Вентиляция корпуса отводит тепло, предотвращая перегрев фасада гарнитура.

Защита от детей блокирует управление кодом, индикатор остаточного тепла показывает доступность камеры.

Сетевой кабель 1,2 м требует линии 16А для стабильной работы.

Ключевые преимущества

12 режимов до 300°C с конвекцией и грилем готовят профессионально, телескопические направляющие выдвигаются на 50 см без ожогов

Гидролизная очистка + эмаль Crystal Clean размягчают жир паром, съемная дверца из 3 стекол моется без химии за 25 минут

Утапливаемые регуляторы + сенсорный дисплей с таймером/часами, доводчик и температура фасада max 65°C обеспечивают безопасность
