Обширная камера 70 литров с шестью позициями размещения открывает возможности для сложных многоярусных блюд, балансируя тепловые потоки равномерно.

Тройное остекление дверцы блокирует передачу жара наружу, обеспечивая безопасный периметр при нагрузках до 280°C.

Каталитические поверхности стенок расщепляют загрязнения при нагреве, минимизируя ручной уход.

Кортинг OKB 3810 FGGR предоставляет десять режимов нагрева с турбоконвекцией и функцией расстойки при 40°C для профессиональной выпечки.

Сенсорная панель мгновенно реагирует на жесты с точной калибровкой, синхронизируясь с контроллером стабильностью ±2°C.

Класс A+ оптимизирует работу при 3 кВт, перераспределяя ресурсы интеллектуально.

Телескопические направляющие на двух уровнях выдерживают 25 кг, исключая термические риски.

Металлическая отделка акцентирует статусность.

Автоматические программы пицца и запекание корректируют параметры по загрузке, где сенсоры определяют оптимальные условия автоматически.

Демпфированное закрывание исключает удары, продлевая механизмы.

Турбовентилятор организует потоки для идентичной обработки без смешения запахов, расстойка контролирует влажность точно.

Таймер устанавливает циклы с автоотключением надежно.

Глубина 57 см вписывается в гарнитуры, вес 36 кг стабилизирует при нагрузках.

Контроль качества включает циклические тесты.

Диапазон 220-240 В обеспечивает автономность.

Темно-зеленый с матовой текстурой гармонирует с натуральными поверхностями.
